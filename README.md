# Voya

Your complete travel assistant — discover countries, explore attractions, unlock local etiquette, and plan your entire trip in one place.

![Voya Preview](preview.png)

## Features

- Search and explore any country in the world
- View top attractions with images and details
- Learn local phrases and cultural etiquette
- Interactive map section
- Beautiful, responsive UI

## Tech Stack

- React + TypeScript
- Vite
- Vercel (deployment)
